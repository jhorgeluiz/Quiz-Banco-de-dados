4 - Como inserir um novo registro em uma tabela no MySQL?

A

ADD INTO nome_da_tabela (coluna1, coluna2) VALUES (valor1, valor2);

B

INSERT INTO nome_da_tabela (coluna1, coluna2) VALUES (valor1, valor2);

C

PUT INTO nome_da_tabela (coluna1, coluna2) VALUES (valor1, valor2);
